# LiBro backend

## Dependencies

System dependencies:

- Modern GHC
- LibreOffice
- libexpat1-dev (Ubuntu) / expat-devel (Fedora)
- libbz2-dev (Ubuntu) / bzip2-devel (Fedora)

Haskell dependencies:

```
cabal install --only-dependencies all
```
